Підтягнути всі залежності:

```bash
npm install
```

Для запуску застосунку у режимі розробника:

```bash
npm run dev
```

Для запуску застосунку в продакшені:

```bash
npm run build
npm run start
```

документація в swagger за посиланням: http://localhost:port/api
